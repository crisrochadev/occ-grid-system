Um sistema de grid simples e funcional para seus projetos web.

3 opções de uso

Mobile First

Comece desenvolvendo para celular, as classes padrões abrangem todos os dispositivos, porém para diferenciar uma tela de outra, use sm,md,lg ou xldependendo da resolução onde o conteúdo será exibido. O OCCGS é versátil te dando a possibilidade de trabalhar com o próprio sistema de grade que foi criado pensando nas necessidades básicas de desenvolvimento, mas também inclui o grid layout e flexbox do próprio css facilitando ainda mais o processo de criação !

Super Leve

Carrege apenas o necessário. O OCCGS é apenas um arquivo css, cuidadosamente escrito para não pesar em seu site ou aplicativo. É possivel usar e editar a versão completa, porém para o melhor desempenho recomenda-se usar a versão compactada. Os componentes e adcionais do sistema de grid, ainda estão em desenvolvimento, mas sempre pensando na melhor performance.

Três formas de uso

Você pode usar a forma mais simples do sistema de grid, mas pode também aproveitar as classes pré prontas com o grid layout e flexbox

OCC Grid System

Simples e funcional, o occgs veio para facilitar sua vida, permitindo que a estrutura seja bem formatada antes mesmo de chegar no css.

Grid Layout

Se você é do team grid, aproveite as classes prontas do occgs e monte seus grids dentro do próprio html, deixando para o css apenas a estilização.

Flexbox

O flexbox também pode ser usado facilmente, 80% das funções já preestabelecidas no sistema de grid, salvo aquelas que precisam de uma atencão especial.
