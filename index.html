    Grid Layout

    Também é possivel criar grids com o grid layout do css, com as classes já estabelecidas, facilitando ainda mais o desenvolvimento das páginas web.

    Estrutura

    [ container principal ]

    grid grid-tc-2 grid-tr-2
    grid-col-1

    [ filhos ]

    grid-col-2

    [ filhos ]

    !

    Funciona como o próprio grid layout, apenas com a facilidade de adcionar a estrutura no próprio html com o uso das classes.

    Forma de uso

    Uso das classes e seus respectivos estilos css

    grid [ container principal ]

    display:grid;

    grid-tc-N [ container principal ]

    grid-template-columns: repeat(N,1fr);

    grid-tr-N [ container principal ]

    grid-template-rows: repeat(N,1fr);

    grid-gap-N [ filhos ]

    grid-gap:Npx;

    grid-col-N [ filhos ]

    grid-column:N;

    grid-row-N [ filhos ]

    grid-row:N;

    grid-col-N-N [ filhos ]

    grid-column:N/N;

    grid-row-N-N [ filhos ]

    grid-row:N/N;

  • Estas são apenas algumas formas de uso, consulte a Documentação
    • !

    Assim como o occgs também é possivel usar sm | md | lg para diferenciar os dispositivos.

    Flexbox

    Use o flexbox do css em seus projetos de forma facil com as classes pré-estabelecidas do occgs.

    Estrutura

    [ container principal ]

    flex j-center a-end
    order-1

    [ filhos ]

    order-2

    [ filhos ]

    !

    Novamente o funcionamento é como o próprio flexbox,bastando apenas adicionar as classes nos elementos html.

    Forma de uso

    Uso das classes e seus respectivos estilos css

    flex [ container principal ]

    display:flex;

    j-center [ container principal ]

    justify-content:center

    a-center[ container principal ]

    align-items:center;

    flex-wrap[ container principal ]

    flex-wrap:wrap;

    dir-col[ container principal ]

    flex-direction:column;

    order-N[ filhos ]

    order:N;

    flex-grow-N[ filhos ]

    flex-grow:N;

  • Estas são apenas algumas formas de uso, consulte a Documentação
    • !

    Assim como o occgs também é possivel usar sm | md | lg para diferenciar os dispositivos.

    Adicionais

    Os Adcionais são as classes de alinhamento e medidas, como [ margin, padding e width e height ]

    Estrutura

    w-10 pt-5 pb-1
    ml-1 mr-05 w-5
    ml-05 mr-1 w-5
    !

    Use as classes para definir os alinhamentos e medidas dos seus elementos dentro do próprio html.

    Forma de uso

    Uso das classes e seus respectivos estilos css

    w-N

    width:N%

    h-Nvh

    height:Nvh;

    m-N

    margin:Npx;

    p-N

    padding:Npx;

    mx-N

    margin-left:Npx;margin-right:Npx;

    my-N

    margin-top:Npx;margin-bottom:Npx;

    pl-N

    padding-left:Npx;

  • Estas são apenas algumas formas de uso, consulte a Documentação
    • !

    Pode-se usar sm | md | lg para diferenciar os dispositivos. As medias vão até 100px, 100%, 100vw ou 100vh.
